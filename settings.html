Thought buffer
Memory tiers (recollection) Sealed arcs are condensed into short "recollections" (tier 2) injected above the live window. Extraction runs in Autonomous mode.
Knowledge graph Entities & relationships (tier 3) merged from arcs; a small subgraph around the current scene is injected. Built in Autonomous mode; injection itself uses no LLM.
Deep extraction & drift Validates extracted facts against a strict entity allow-list (anti-hallucination), scores each arc's significance (important arcs pin, filler fades), and flags contradictions for review. Runs in Autonomous mode; flags appear in Memory → Drift.
Context budget One hard token ceiling over ALL injected memory (buffer + favorites + recollections + graph + ToC). Fills by tier priority; on overflow it condenses (drops whole lines, never mid-sentence) and lowest-priority tiers drop first. Pure code, no LLM. Off → each tier uses its own budget with no global cap. See Memory → Memory budget for the health bar.
Two-stage pipeline (§4b) Stage-1 memory engine: the cheap agent assembles the memory core (recollections + graph + retrieval) and hands the writer (your normal model) only a distillate. The core is cached per scene — on a static scene it is reused and only the cheap buffer/favorites refresh, saving work. Off → injection behaves exactly as before (fresh build every turn). Enabled automatically in Autonomous.
Resurfacing
Arcs & auto-hide Old messages are sealed into arcs and hidden in slabs to keep the active window small.
Timeline & activity Memory → Timeline shows one chronological feed: background work (arc seals, extraction, graph merges, drift, audits) plus restore points from Backups, each with a ⏪ Restore. No LLM, no lorebook writes — just bookkeeping.
Backups
Catch-up backfill For chats where the extension was enabled late: segment the prior history into arcs (Light) or process them with the cheap AI (Full → recollections + graph + lorebook entries). One-time; this offer auto-clears once memory grows forward.
